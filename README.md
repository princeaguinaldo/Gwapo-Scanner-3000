# 💫 GWAPO SCANNER 3000 — Python

GWAPO SCANNER 3000 is a fun Python program that uses your webcam and OpenCV to detect faces in real time and “scan” them.

---

## 📦 Requirements

Install the required Python packages:

```bash
pip install opencv-python numpy
```

---

## ▶️ How to Run

Make sure your webcam is connected, then run:

```bash
python gwapo_scanner.py
```

---

## 🎮 Controls

| Key    | Action                |
|--------|-----------------------|
| SPACE  | Scan the current face |
| Q      | Quit the program      |

---

## 📷 Features

- Real-time webcam feed  
- Face detection using OpenCV  
- Simple keyboard controls  
- Lightweight and easy to run  

---

## ⚠️ Notes

- Ensure your camera permissions are enabled  
- Works best in good lighting conditions  
- Close other apps that may be using the webcam
